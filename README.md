# Description

Code to 
1. simulate hypothetical studies illustrating time-related biases (Figurexx_simulation.R)
3. emulate a target trial on covid-19 vaccination in the outcome of patients with NSCLC or stage IV melanoma treated with ICI from the data available here : https://www.nature.com/articles/s41586-025-09655-y. The analyses can be reproduced by running the files main_nlscc.R and main_melanoma.R
